This program is used for Emotion Detection through facial expressions using
Convolution Neural Networks developed using Torch7 framework.

This module also makes use of OpenCV for taking realtime input from
the webcam/camera.

To run this program, set up proper paths for directories and files.
Then just simply run the lua script - run.lua present in this folder:

	qlua run.lua
